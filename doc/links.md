## Links

### Mandelbrot online explorers

This is a list of sites where you can explore Mandelbrot Set. Some of them let you easily get z real and imaginary parts and maximum number of iterations so you can use them with this program to try to reproduce the same result.

1. [mandel.gart.nz](http://mandel.gart.nz/)
2. [math.hws.edu](http://math.hws.edu/eck/js/mandelbrot/MB.html)
3. [guciek.github.io](https://guciek.github.io/web_mandelbrot.html)
4. [xaos-project.github.io](http://xaos-project.github.io/XaoSjs/)
5. [sciencedemos.org.uk](https://sciencedemos.org.uk/mandelbrot.php)
